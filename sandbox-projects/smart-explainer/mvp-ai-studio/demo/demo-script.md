# Smart Explainer Demo Script

## Recording Timeline (Total: ~160 seconds / 2:40)

### 0–08s: Title card
- Title: "Smart Explainer — MVP"
- Tagline: "Gemini 3 explains errors & screenshots"
- Show project logo or simple title slide

### 08–25s: Text Input Demo - Java NullPointerException
- Paste `java-nullpointer.txt` into text area
- Click "Explain" button
- Show loading state briefly

### 25–60s: Show Text Output
- Display the output sections:
  - **Summary**: One-line explanation
  - **Likely causes**: 2-3 bullet points
  - **Diagnostic steps**: Actionable commands/checks
  - **Suggested fix**: Code snippets or commands
  - **Confidence**: Score with reasoning
- Scroll through each section to show completeness

### 60–95s: Image Input Demo
- Switch to "Image" tab
- Upload `stacktrace-screenshot.png`
- Click "Explain" button
- Show visual description + analysis

### 95–140s: JSON Mode Toggle
- Show the same result in structured JSON format
- Toggle between human-readable and JSON view
- Highlight the clean JSON structure
- Show how it can be consumed programmatically

### 140–160s: Closing
- Quick recap: "Gemini 3 features used:"
  - Step-by-step reasoning
  - Vision capabilities (image analysis)
  - Structured output (JSON mode)
- Display public demo link
- Call to action: "Try it yourself at [your-link]"

## Recording Tips

1. **Screen Setup**: Clear desktop, close unnecessary windows
2. **Browser**: Full screen or clean browser window
3. **Cursor**: Use cursor highlighting if available
4. **Pacing**: Smooth transitions, no rushed clicks
5. **Audio**: Optional voiceover or background music
6. **Export**: 1080p, MP4 format, ≤ 3 minutes

## Backup Scenarios

If primary demo fails:
- Have `port-in-use.txt` ready as backup text input
- Have `terminal-error.png` ready as backup image
- Pre-record sections if live demo is risky
