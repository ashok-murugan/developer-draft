## 🎯 What This App Does

### Input Options:
- 📝 **Text**: Error messages, logs, technical descriptions
- 🖼️ **Images**: Screenshots, terminal errors, diagrams

### Output Provides:
1. ✨ **One-line summary** of the issue
2. 🔍 **Root cause analysis** (2-3 likely causes)
3. 🛠️ **Diagnostic steps** with exact commands
4. 💡 **Suggested fixes** with code snippets
5. 📊 **Confidence score** (0.0-1.0) with reasoning

### Special Features:
- 🔄 **JSON mode** for structured output
- 👁️ **Vision support** for screenshot analysis
- 🎯 **High accuracy** with confidence scoring
- ⚡ **Fast responses** (< 5 seconds)
